Keystroke Logging System (Keylogger) 

This project is a safe and educational keylogger built using Java Swing. It captures keystrokes only inside the application window and logs them into a file along with the date and time of each key press.

The goal of this application is to help learners undersatand:
  How keystrokes are detected in Java,
  How GUI-based applications handle keyboard events,
  How to save logs to a file,
  How timestamps are applied for tracking events,
  Why ethical programming matters in cybersecurity.

  Features of this Project includes,
   -> Application-level keylogging,
   -> Timestamped logging,
   -> Lightweight GUI,
   -> Real-time file generation,
   -> Safe & ethical.
